# eLearning Project
***M&Y Yazılım Akademi Full Stack .Net Development - Resume with MVC eğitim projesi.***

# Proje Hakkında
***Asp.NET MVC ve Entity Framework DbFirst yaklaşımı ile bir CV sitesi geliştirdik. Admin Panel yönetimi,
projelerin listelenmesi ve ziyaretçilerin mesaj gönderebildiği bir yapı üstüne çalıştık.***

# Kullanılan Teknolojiler
***ASP.NET - EntityFramework - DB First - SQL Server - MVC - Bootstrap***
